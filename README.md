# Movie Explorer

Movie Explorer is a web application built with **React.js**, **JavaScript**, and **CSS**, that allows users to browse, search, and save their favorite movies.  
It uses data fetched from [The Movie Database (TMDB)](https://www.themoviedb.org/) API.

---

## Features

- Browse popular and trending movies  
- Search for movies by title  
- View detailed movie information (poster, overview, rating, release date, etc.)  
- Save and manage favorite movies locally  
- Responsive UI for both desktop and mobile devices  

---

## Tech Stack

- **Frontend:** React.js  
- **Styling:** CSS  
- **Data Source:** [The Movie Database (TMDB) API](https://www.themoviedb.org/documentation/api)  

---

## Installation & Setup
Clone the repository
   ```bash
   git clone https://github.com/<your-username>/<your-repo-name>.git
   cd <your-repo-name>

Install dependencies
npm install


Create an .env file in the project root and add your TMDB API key:
API_KEY=your_api_key_here


Run the development server
npm run dev


Open http://localhost:3000
 to view it in the browser.
